# 電気電子回路1 総合演習アプリ

電気電子回路1の演習問題を学習するためのWebアプリケーションです。
スマホやPCのブラウザで手軽に「一問一答」や「暗記学習」ができます。

## 📁 ディレクトリ構成

```text
.
├── index.html          # アプリの入口（これをブラウザで開く）
├── style.css           # デザイン設定
├── script.js           # アプリの動作ロジック
├── data.js             # 【重要】問題データはここにあります
├── generate_files.py   # 個別の演習HTMLを生成するPythonスクリプト
└── excasize/           # 個別の演習ファイル格納フォルダ
    ├── ex01.html
    ├── ex02.html
    ...
    └── ex07.html
```

## 🚀 使い方

1. `index.html` をブラウザ（Chrome, Safari, Edgeなど）で開いてください。
2. 以下の2つのモードで学習できます。
   - **一覧モード**: 答えが隠されており、タップして確認できます。
   - **一問一答モード**: クイズ形式でランダムに出題されます。

## 🛠 誤植・間違いの修正について (Pull Request)

問題文や解答に誤り（誤植、不適切な解説など）を見つけた場合は、**Pull Request (PR)** を送っていただけると大変助かります！

### 修正の手順

問題データはすべて **`data.js`** というファイルに集約されています。

1. このリポジトリを **Fork** してください。
2. **`data.js`** を開き、該当する問題の箇所を修正してください。
   ```javascript
   // 例: data.js
   { 
     id: "01-1-a", 
     cat: "01", 
     q: "正しい問題文に修正...", // ←ここを直す
     opts: [...], 
     ans: 1 
   },
   ```
3. 修正内容をコミットし、**Pull Request** を作成してください。
4. 内容を確認し、マージさせていただきます。

※ `excasize/` フォルダ内のファイルや `generate_files.py` は、`data.js` の内容とは独立している場合がありますが、Webアプリ版（`index.html`）への修正を優先するため、まずは `data.js` の修正をお願いします。
